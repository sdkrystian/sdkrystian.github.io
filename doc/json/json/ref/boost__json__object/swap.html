object::swap

Swap the contents.

Synopsis 
void
swap(
    object& other);
Description

Exchanges the contents of this object with another object. Ownership of the respective storage objects is not transferred.

  • If *other.storage() == *sp, ownership of the underlying memory is swapped in constant time, with no possibility of exceptions. All iterators and references remain valid.
  • If *other.storage() != *sp, the contents are logically swapped by making copies, which can throw. In this case all iterators and references are invalidated.
Complexity

Constant or linear in size() plus other.size().

Exception Safety

Strong guarantee. Calls to storage::allocate may throw.

Parameters

Name

Description

other

The object to swap with.
