object::object (10 of 11 overloads)

Constructor.

Synopsis 
object(
    std::initializer_list< std::pair< key_type, value_ref >> init,
    storage_ptr sp = {});
Description

The object is constructed with a copy of the values in the initializer-list in order, using the specified storage. If multiple elements in the range have keys that compare equivalent, only the first occurring key will be inserted.

Complexity

Linear in init.size().

Exception Safety

Strong guarantee. Calls to storage::allocate may throw.

Parameters

Name

Description

init

The initializer list to insert.

sp

A pointer to the storage to use. The container will acquire shared ownership of the storage object.
