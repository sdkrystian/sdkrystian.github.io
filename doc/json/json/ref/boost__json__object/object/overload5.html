object::object (5 of 11 overloads)

Move constructor.

Synopsis 
object(
    object&& other);
Description

The object is constructed by acquiring ownership of the contents of other and shared ownership of the storage of other.

Remarks

After construction, the moved-from object behaves as if newly constructed with its current storage.

Complexity

Constant.

Exception Safety

No-throw guarantee.

Parameters

Name

Description

other

The object to move.
