object::insert (3 of 3 overloads)

Insert elements.

Synopsis 
void
insert(
    std::initializer_list< std::pair< key_type, value_ref >> init);
Description

The elements in the initializer list are inserted at the end, in order. If multiple elements in the range have keys that compare equivalent, only the first occurring key will be inserted.

Complexity

Linear in init.size().

Exception Safety

Strong guarantee. Calls to storage::allocate may throw.

Parameters

Name

Description

init

The initializer list to insert
