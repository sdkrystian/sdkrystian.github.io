number_cast (2 of 2 overloads)

Returns a value converted to a numeric type.

Synopsis

Defined in header <boost/json/number_cast.hpp> 

template<
    class T>
T
number_cast(
    value const& jv);
Description

This function attempts to convert a value to the numeric type T using these rules:

  • If ! jv.is_number(), an exception is thrown.
  • If T is an integral type and the value can be represented exactly, it is returned. Otherwise an exception is thrown.
  • If T is a floating point type, the value returned is either exact or the closest approximation which is determined as-if using static_cast<T>.
Constraints
