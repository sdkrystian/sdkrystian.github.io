basic_parser::on_string_part

Called with characters corresponding to part of the current string.

Synopsis 
void
on_string_part(
    string_view s,
    error_code& ec);
Description

This function is called during parsing to provide the characters corresponding to a string value being parsed. More string data may be provided by zero or more calls to on_string_part, followed by a final call to on_string.

Parameters

Name

Description

s

The string view holding the next buffer of string data, with escapes converted to their actual characters.

ec

The error, if any, which will be returned by the current invocation of write_some, write, or finish.
