basic_parser::on_key

Called with the last characters corresponding to the current key.

Synopsis 
void
on_key(
    string_view s,
    error_code& ec);
Description

This function is called during parsing to provide the final character string corresponding to the key being parsed as part of an object. The key is considered complete with these characters.

Parameters

Name

Description

s

The string view holding the final buffer of key data, with escapes converted to their actual characters.

ec

The error, if any, which will be returned by the current invocation of write_some, write, or finish.
