basic_parser::on_document_end

Called when the JSON parsing is done.

Synopsis 
void
on_document_end(
    error_code& ec);
Description

This function is invoked once when a complete JSON document has been parsed without errors. The call happens last; no other calls are made before the parser is reset.

Parameters

Name

Description

ec

The error, if any, which will be returned by the current invocation of write_some, write, or finish.
