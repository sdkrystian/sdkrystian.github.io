basic_parser::on_array_end

Called when the end of the current array is encountered.

Synopsis 
void
on_array_end(
    error_code& ec);
Description

This function is called during parsing after all of the elements of an array have been parsed, and the closing bracket for the array is reached.

Parameters

Name

Description

ec

The error, if any, which will be returned by the current invocation of write_some, write, or finish.
