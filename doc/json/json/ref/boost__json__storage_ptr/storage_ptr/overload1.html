storage_ptr::storage_ptr (1 of 4 overloads)

Default constructor.

Synopsis 
storage_ptr();
Description

This constructs a default storage pointer. The default storage is not reference counted, uses global operator new and delete to obtain memory, and requires calls to deallocate.

Complexity

Constant.

Exception Safety

No-throw guarantee.
