storage_ptr::operator= (1 of 2 overloads)

Move assignment.

Synopsis 
storage_ptr&
operator=(
    storage_ptr&& other);
Description

Shared ownership of the storage owned by *this is released, and shared ownership of the storage owned by other is acquired by move construction. After construction, the moved-from object will point to the default storage.

Complexity

Constant.

Exception Safety

No-throw guarantee.

Parameters

Name

Description

other

The storage pointer to move.
