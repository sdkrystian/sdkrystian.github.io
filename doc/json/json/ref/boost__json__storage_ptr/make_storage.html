storage_ptr::make_storage

Create a new, counted storage object and return a pointer to it.

Synopsis

Defined in header <boost/json/storage_ptr.hpp> 

template<
    class U,
    class... Args>
friend storage_ptr
make_storage(
    Args&&... args);
Description

This functions similarly to make_shared.

Mandates 
is_storage<T>::value == true
Complexity

Same as T(std::forward<Args>(args)...).

Exception Safety

Strong guarantee.

Parameters

Name

Description

args

Parameters forwarded to the constructor of T.

Template Parameters

Type

Description
