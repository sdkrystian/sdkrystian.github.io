has_to_value

Determine if T can be converted to a JSON value.

Synopsis

Defined in header <boost/json/to_value.hpp> 

template<
    class T>
using has_to_value = see-below;
Description

This template type alias is std::true_type when one or more of the following are true:

  • value is constructible with the signature (T, storage_ptr)
  • A public non-static member function T::to_json( value& jv ) const exists.
  • A specialization of to_value_traits exists which contains the public static member function to_value_traits<T>:: assign( value&, T const&).
  • The type T matches one of the generic types supported by the library.

Otherwise, this template type alias is std::false_type.

Convenience header <boost/json.hpp>
