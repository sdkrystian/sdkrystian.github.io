parse (1 of 2 overloads)

Parse a string of JSON.

Synopsis

Defined in header <boost/json/parser.hpp> 

value
parse(
    string_view s,
    error_code& ec,
    storage_ptr sp = {});
Description

This function parses an entire single string in one step to produce a complete JSON object, returned as a value. If the buffer does not contain a complete serialized JSON, an error occurs. In this case the returned value will be null, using the default storage.

Complexity

Linear in s.size().

Exception Safety

Strong guarantee. Calls to storage::allocate may throw.

Parameters

Name

Description

s

The string to parse.

ec

Set to the error, if any occurred.

sp

The storage that the new value and all of its elements will use. If this parameter is omitted, the default storage is used.

Return Value

A value representing the parsed JSON, or a null if any error occurred.
