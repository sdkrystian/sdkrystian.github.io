parser::reserve

Reserve internal storage space.

Synopsis 
void
reserve(
    std::size_t n);
Description

This function reserves space for n bytes in the parser's internal temporary storage. The request is only a hint to the implementation.

Exception Safety

No-throw guarantee.

Parameters

Name

Description

n

The number of bytes to reserve. A good choices is C * sizeof(value) where C is the total number of value elements in a typical parsed JSON.
