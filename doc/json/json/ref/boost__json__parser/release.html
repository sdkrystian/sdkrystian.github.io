parser::release

Return the parsed JSON as a value.

Synopsis 
value
release();
Description

If is_done() returns true, then the parsed value is returned. Otherwise an exception is thrown.

Exceptions

Type

Thrown On

std::logic_error

! is_done()

Return Value

The parsed value. Ownership of this value is transferred to the caller.
