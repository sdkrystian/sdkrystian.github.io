value::get_array (1 of 2 overloads)

Return a reference to the underlying array, without checking.

Synopsis 
array&
get_array();
Description

This is the fastest way to access the underlying representation when the kind is known in advance.

Preconditions 
this->is_array()
Complexity

Constant.

Exception Safety

No-throw guarantee.
