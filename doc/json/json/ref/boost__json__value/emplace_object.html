value::emplace_object

Return a reference to an object, changing the kind and replacing the contents.

Synopsis 
object&
emplace_object();
Description

The value is replaced with an empty object using the current storage, destroying the previous contents.

Complexity

Linear in the size of *this.

Exception Safety

No-throw guarantee.
