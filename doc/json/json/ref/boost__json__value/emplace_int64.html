value::emplace_int64

Return a reference to a std::int64_t, changing the kind and replacing the contents.

Synopsis 
std::int64_t&
emplace_int64();
Description

The value is replaced with a std::int64_t initialized to zero, destroying the previous contents.

Complexity

Linear in the size of *this.

Exception Safety

No-throw guarantee.
