value::emplace_array

Return a reference to an array, changing the kind and replacing the contents.

Synopsis 
array&
emplace_array();
Description

The value is replaced with an empty array using the current storage, destroying the previous contents.

Complexity

Linear in the size of *this.

Exception Safety

No-throw guarantee.
