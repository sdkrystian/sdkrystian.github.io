value::value (12 of 34 overloads)

Construct an array.

Synopsis 
value(
    array other);
Description

The value is constructed from other, using the same storage. To transfer ownership, use std::move:

Example 
array arr( {1, 2, 3, 4, 5} );

// transfer ownership
value jv( std::move(arr) );

assert( arr.empty() );
assert( *arr.storage() == *jv.storage() );
Complexity

Constant.

Exception Safety

No-throw guarantee.

Parameters

Name

Description

other

The array to construct with.
