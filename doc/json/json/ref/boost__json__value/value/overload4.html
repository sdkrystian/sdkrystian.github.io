value::value (4 of 34 overloads)

Copy constructor.

Synopsis 
value(
    value const& other);
Description

The value is constructed with a copy of the contents of other, using the storage of other.

Complexity

Linear in the size of other.

Exception Safety

Strong guarantee. Calls to storage::allocate may throw.

Parameters

Name

Description

other

The value to copy.
