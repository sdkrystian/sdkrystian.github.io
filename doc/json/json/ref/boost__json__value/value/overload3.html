value::value (3 of 34 overloads)

Pilfer constructor.

Synopsis 
value(
    pilfered< value > other);
Description

The value is constructed by acquiring ownership of the contents of other using pilfer semantics.

Remarks

After construction, the moved-from value may only be destroyed.

Complexity

Constant.

Exception Safety

No-throw guarantee.

Parameters

Name

Description

other

The value to pilfer.

See Also

pilfer Pilfering constructors are described in Valueless Variants Considered Harmful, by Peter Dimov.
