value::value (21 of 34 overloads)

Construct a string.

Synopsis 
value(
    string_kind_t,
    storage_ptr sp = {});
Description

This is the fastest way to construct an empty string, using the specified storage. The variable string_kind may be passed as the first parameter to select this overload:

Example 
// Construct an empty string

value jv( string_kind );
Complexity

Constant.

Exception Safety

No-throw guarantee.

Parameters

Name

Description

sp

A pointer to the storage to use. The container will acquire shared ownership of the storage object.

See Also

string_kind
