array::array (5 of 11 overloads)

Constructor.

Synopsis 
template<
    class InputIt>
array(
    InputIt first,
    InputIt last,
    storage_ptr sp = {});
Description

The array is constructed with the elements in the range {first, last), preserving order, using the specified storage.

Constraints 
std::is_constructible_v<value, std::iterator_traits<InputIt>::value_type>
Complexity

Linear in std::distance(first, last)

Exception Safety

Strong guarantee. Calls to storage::allocate may throw.

Parameters

Name

Description

first

An input iterator pointing to the first element to insert, or pointing to the end of the range.

last

An input iterator pointing to the end of the range.

sp

A pointer to the storage to use. The container will acquire shared ownership of the storage object.

Template Parameters

Type

Description

InputIt

a type meeting the requirements of InputIterator.
