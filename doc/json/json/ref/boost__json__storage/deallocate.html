storage::deallocate

Deallocate memory.

Synopsis 
void
deallocate(
    void* p,
    std::size_t n,
    std::size_t align = alignof(max_align_t));
Description

Deallocates the storage pointed to by p.

Preconditions

pwas returned by a prior call to u.allocate( bytes, align ) where this->is_equal(u), and the storage p points to was not previously deallocated.
