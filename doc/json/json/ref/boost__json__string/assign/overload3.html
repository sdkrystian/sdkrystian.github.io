string::assign (3 of 10 overloads)

Assign characters to a string.

Synopsis 
string&
assign(
    string const& other,
    std::size_t pos,
    std::size_t count);
Description

Replace the contents with a copy of the substring {pos, pos+count) of other. If the requested substring lasts past the end of the string, or if count == npos, the resulting substring is {pos, other.size()).

Complexity

Linear in count.

Exception Safety

Strong guarantee.

Remarks

Calls to storage::allocate may throw.

Return Value

*this

Parameters

Name

Description

other

The string to use as a source to copy from.

pos

The starting character position to copy from.

count

The number of characters to copy.

Exceptions

Type

Thrown On

std::out_of_range

pos >= other.size().
