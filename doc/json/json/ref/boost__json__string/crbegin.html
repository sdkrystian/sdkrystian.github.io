string::crbegin

Return a reverse iterator to the first character of the reversed container.

Synopsis 
const_reverse_iterator
crbegin() const;
Description

Returns the pointed-to character that corresponds to the last character of the non-reversed container.

Complexity

Constant.

Remarks

If the container is empty, the returned iterator is equal to rend()
