string::swap

Swap the contents.

Synopsis 
void
swap(
    string& other);
Description

Exchanges the contents of this string with another string. Ownership of the respective storage objects is not transferred.

  • If *other.storage() == *sp, ownership of the underlying memory is swapped in constant time, with no possibility of exceptions. All iterators and references remain valid.
  • If *other.storage() != *sp, the contents are logically swapped by making a copy, which can throw. In this case all iterators and references are invalidated.
Complexity

Constant or linear in size() plus other.size().

Precondition 
&other != this
Remarks

Calls to storage::allocate may throw.

Exception Safety

Strong guarantee.

Parameters

Name

Description

other

The string to swap with
